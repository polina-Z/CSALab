1. Устанавливать аргументы можно с помощью команды set args <arguments>.
2. Для создания точки останова используется команда break(можно использовать просто b) func_name(or string number).
3. Чтобы выполнить следующую строку кода в программе можно использовать команду next(n) - шаг с обходм или step(s) - шаг с заходом.
4. Чтобы выполнить код внутри функции сразу, используется команда next(шаг с обходом). Чтобы отладить код внутри функции, используется  команда step(шаг с заходом).
5. Чтобы продолжить выполнение программы после остановки на точке останова, используется команда continue
6. Вывести значение переменных или выражений позволяет команда print(arguments)
7. Чтобы отобразить значение переменной после каждого шага, можно использовать команду display(arguments)
8. Информация об аргументах функции - info args, локальных переменных - info locals
9. Для выхода из GDB используется команда quit(q).
